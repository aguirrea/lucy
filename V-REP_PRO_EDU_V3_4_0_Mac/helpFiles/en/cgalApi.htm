CGAL Plugin API reference

The list of API functions below represent an interface to the CGAL library. Currently only one function is supported. The source code to the plugin can be found in programming/v_repExtSurfaceReconstruction.

simExtSurfaceReconstruction_reconstruct

Description Constructs a shape based on a point cloud.
Lua synopsis number shapeHandle=simExtSurfaceReconstruct(number pointCloudHandle, number algorithm)
Lua parameters
pointCloudHandle: a handle to a point cloud.
algorithm: a reconstruction algorithm. Currently, only sim_surfacereconstruction_algorithm_scalespace is supported.
Lua return values
shapeHandle: the handle of the resulting shape, or -1 in case of an error



Constants

Reconstruction algorithms

sim_surfacereconstruction_algorithm_scalespace
sim_surfacereconstruction_algorithm_poisson
sim_surfacereconstruction_algorithm_advancingfront
